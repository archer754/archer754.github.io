    【面试】手撕call、apply、bind

    vuePress-theme-reco Ian Chen    2017 - 2021

    【面试】手撕call、apply、bind

    Ian Chen 2021-6-9 中级高级

    # 1. call()

    # 用法：

    function.call(thisArg,arg1,arg2...){ }
    1

    功能：使用一个指定的this值和单独给出的一个或多个参数来调用一个函数

    返回值：使用调用者提供的this值和参数调用该函数的返回值。若该方法没有返回值，则返回undefined

    1.this参数可以传null或者undefined，此时this指向window
    2.this参数可以传基本类型数据，原生的call会自动用Object()转换
    3.函数是可以有返回值的
    4.降函数添加到指定对象中，并用delete删除（消除副作用）

    小试牛刀

    function method(val1,val2){
    return this.a + this.b + val1 + val2;
}
const obj = {
    a: 1,
    b: 2
}
console.log(method.call(obj,3,4)); // 10

    # 实现：

    1.获取第一个参数（注意第一个参数为null或undefined时，this指向window），构建对象
2.将对应函数传入该对象中
3.获取参数并执行相应函数
4.删除该对象中函数，消除副作用
5.返回结果

    Function.prototype.myCall = function(context,...args){
    //获取第一个参数（注意第一个参数为null或undefined时，this指向window），构建对象
    context = context ? Object(context) : window;

    //将对应参数传入该对象中
    context.fn = this;

    //获取参数并执行相应函数
    let result = context.fn(...args);

    //删除该对象中函数，消除副作用
    delete context.fn;

    //返回结果
    return result
}
//.....
console.log(method.myCall(obj, 3, 4)); // 10

    # 2. apply()

    # 用法：

    function.apply(thisArg,[argsArray])
    1

    功能：调用一个具有给定this值的函数，以及作为一个数组（或类似数组对象）提供的参数

    返回值：调用有指定this值的函数的结果

    小试牛刀

    function method(val1,val2){
    return this.a + this.b + val1 + val2;
}
const obj = {
    a: 1,
    b: 2
}
console.log(method.apply(obj, [3, 4] )); // 10

    # 实现

    apply和call的区别主要是参数不同，所有实现步骤和call类似

    Function.prototype.myApply = function (context, arr){
    context = context ? Object(context) : window;
    context.fn = this;

    let result = arr ? context.fn(...arr) ? context.fn();

    delete context.fn;

    return result;
}
//...
console.log(method.myApply(obj, [3, 4])); // 10

    # 3. bind()

    # 用法：

    function.bind(thisArg[, arg1[, arg2[, ...]]]);
    1

    功能：创建一个心的函数，在bind()被调用时，这个心函数的this被指定为bind()的第一个参数，而其余参数座位心函数的参数，供调用时使用 返回值：返回一个原函数的拷贝，并拥有指定的this值和初始参数

    1.多次bind绑定无效，只会绑定在第一个上面
    2.bind之后，其prototype为undefined
    3.bind()返回值是改造参数后的原函数的拷贝，所有调用bind()的必须是函数

    小试牛刀

    function method(val1,val2){
    return this.a + this.b + val1 + val2;
}
const obj = {
    a: 1,
    b: 2
}
const bindMethod = method.bind(obj, 3, 4);
console.log(bindMethod()); // 10

    # 实现

    1.能够改变this指向
2.返回的是一个函数
3.能够接受多个参数
4.支持柯里化形式传参fun(arg1)(arg2)
5.获取到调用bind()返回值后，若使用new调用（当作构造函数），bind()传入的上下文context失效

    Function.prototype.myBind = function (context, ...args){
    if(typeof(this) !== 'function'){
        throw new TypeError('The bound onject needs to be a function');
    }

    const self = this;
    //定义一个中装函数
    const fNOP = function(){};
    const fBound = function(...fBoundArgs){
        //利用apply改变this指向
        //接受多个参数+支持柯里化形式传参
        //返回值通过new调用时，this指向当前实例，（因为this是当前实例，实例的隐式原型上有fNOP的实例（fnop）；fnop instanceof fNOP 为true）
        return self.apply(this instanceof fNOP ? this : context, [...args, ...fBoundArgs]);
    }

    //将调用函数的原型赋值到中转函数的原型上
    if(this.prototype){
        fNOP.prototype = this.prototype;
    }
    //通过原型的方式继承调用函数的原型
    fBound.prototype = new fNOP();

    return fBound;
}
